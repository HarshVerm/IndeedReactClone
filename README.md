<h2>Getting Started</h2>

Clone the repository, Do npm install and run mock server on --port 8000

<h2>TECH STACK USED:</h2>
- HTML
- CSS
- Javascript
- React JS

<h2>TECH LIBRARY USED:</h2>
- React APP
- Redux
- React Router
- Axios
- Material UI

## Usage

```bash
npm start
json-server --watch db.json --port 8000
```
